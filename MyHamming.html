MyHamming module¶

MyHamming.py - Module de correction d’erreurs par code de Hamming.

Ce module fait partie du projet DataMorse. Il permet d’encoder et de décoder des matrices binaires avec contrôle de parité, et de corriger certaines erreurs simples ou doubles via redondance.

Auteur : Lucas BATTAGLIA

Version : v1.2

class MyHamming.MyHamming(matrix1, matrix2=None)¶

Bases : object

Classe pour encoder et décoder des matrices en utilisant un code de Hamming étendu.

Cette classe permet l’encodage avec bits de parité, la détection d’erreurs, et la tentative de correction simple ou double en comparant deux matrices.

Paramètres

  • matrix2 (list[list[int]], optional) – Matrice secondaire pour comparaison (double détection).
