ReedSolomon module#

ReedSolomon.py - Module pour l’encodage et le décodage de messages avec le code Reed-Solomon.

Ce module fait partie du projet DataMorse.

Il permet d’encoder des messages en utilisant le code Reed-Solomon pour la correction d’erreurs, et de décoder les messages reçus tout en corrigeant les erreurs détectables.
