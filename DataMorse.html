DataMorse module¶

DataMorse.py - Module pour le chiffrement et le déchiffrement de messages en utilisant le code Morse.

Ce module est la partie principale du projet DataMorse.

Il permet d’encoder des messages en code Morse, de les transformer en une matrice binaire, et de générer une image représentant ce message. Il inclut également un décodeur pour récupérer le message original à partir de l’image.
